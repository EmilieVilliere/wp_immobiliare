Initialization of Wordpress - Download wp structure on https://wordpress.org/download/ - move it on htdocs for xampp server - create a child theme - create a database on localhost/php-my-admin - and then go on localhost/project_name/wp-admin/setup-config.php - follow the instructions - on the first step : set the database name, set root on username, set no password - on the second : set the same title of your project, an usersname, a password, a mail and then : go ! - log in on go on themes page, activate the child theme and customize it !
